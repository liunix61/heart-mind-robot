# Live2D Cubism SDK

这是 Live2D Cubism SDK 的 Framework 部分，用于在 HeartMindRobot 项目中提供 Live2D 角色渲染和动画功能。

## 目录结构

- `Framework/` - Live2D Cubism SDK 的核心框架库
  - `src/` - 源代码文件
    - `Effect/` - 特效系统（呼吸、眨眼、姿态等）
    - `Id/` - ID 管理系统
    - `Math/` - 数学计算库
    - `Model/` - 模型管理
    - `Motion/` - 动作系统
    - `Physics/` - 物理模拟
    - `Rendering/` - 渲染引擎
    - `Type/` - 基础数据类型
    - `Utils/` - 工具函数
